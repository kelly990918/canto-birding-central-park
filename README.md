# Central Park Cantonese Bird Field Guide

An interactive, single-file web app for identifying birds commonly seen in Central Park, with a full Cantonese interface and practical field information.

**Vibe coded by Grok** (built by xAI) — 2026

## English

### About
This is a lightweight, self-contained bird identification tool focused on species frequently observed in Central Park, New York. It is designed to be practical for field use, with information presented in natural Cantonese.

### Features
- 98 common Central Park bird species
- Categorized using proper Linnaean taxonomy (雁鴨類, 猛禽類, 雀形目, etc.)
- Filter by bird type and migration status
- Search by English, Cantonese, or scientific name
- Each bird card includes:
  - English + Cantonese name
  - Scientific name
  - Key identification features and ecology notes
  - Origin information (native vs introduced/invasive)
  - State bird status where applicable
- Mobile-friendly layout (1 column on phones)
- Works completely offline once the HTML file is downloaded

### How to Use
1. Open the `central-park-birds.html` file in any modern web browser.
2. Use the search bar or click the filter buttons to narrow down results.
3. Click the ℹ️ icon next to “分類（目）” or “遷徙狀態” to view explanations of the categories.

### Tech Stack
- Single HTML file
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- No backend or external dependencies required
