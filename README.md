## 粵語版

一個實用、美觀的中央公園鳥類識別網頁應用，採用完整粵語界面、正確林奈分類系統，同埋用自然香港式粵語撰寫的生態描述。

每張鳥卡都清晰易讀，方便你喺公園即時識別最常見的雀鳥。支援多種篩選同即時搜尋，無論你係新手定資深觀鳥人，都可以輕鬆搵到想要嘅資訊。

### 主要特色
- 收錄 52+ 種中央公園常見鳥類
- 正確分類：雁鴨類、鷺類、猛禽類、鴿類、啄木鳥類、雀形目等
- 可多選篩選器：鳥類目類 + 遷徙狀態（常留鳥 / 夏候鳥 / 冬候鳥 / 過境鳥）
- 強大即時搜尋：支援英文、粵語、學名
- 每張鳥卡有清晰名稱、顏色標籤同生態識別特徵
- 資訊彈窗詳細解釋分類同遷徙術語
- 乾淨現代界面，專為公園即時使用而設計

## English

A practical and beautiful bird identification web app for Central Park, New York. It features a complete Cantonese (粵語) interface, proper Linnaean taxonomy, and rich ecological notes written in natural Hong Kong-style Cantonese.

Each bird card is clear and easy to read, helping you quickly identify common birds while in the park. With flexible filters and real-time search, both beginners and experienced birders can easily find the information they need.

### Key Highlights
- 52+ common Central Park bird species
- Accurate taxonomy categories: 雁鴨類, 鷺類, 猛禽類, 鴿類, 啄木鳥類, 雀形目, etc.
- Multi-select filters by taxonomy + migration status (常留鳥 / 夏候鳥 / 冬候鳥 / 過境鳥)
- Powerful real-time search supporting English, Cantonese, and scientific names
- Clean bird cards with clear names, color-coded badges, and ecology notes
- Info modal explaining taxonomy categories and migration terms
- Clean, modern interface designed for quick use in the field
